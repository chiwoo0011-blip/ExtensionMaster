연세대학교재활학교 내선번호안내를 앱처럼 사용하세요

PC에서 앱으로 설치하기 (Chrome)
1 사이트 접속
Chrome 브라우저에서 내선번호안내 사이트에 접속합니다.
🔒
extension-master.vercel.app
☆ ⬇ 설치
2 설치 버튼 클릭
주소창 오른쪽에 ⬇ 설치 아이콘 또는 모니터+화살표(⊞) 아이콘이 나타납니다. 클릭하세요.
↓
또는 Chrome 메뉴(⋮) → "앱 설치..."를 선택합니다.

새 창

3 설치 확인
"앱을 설치하시겠습니까?" 팝업이 나타나면 "설치" 버튼을 클릭합니다.
4 완료!
바탕화면과 작업표시줄에 앱 아이콘이 생깁니다. 독립 창으로 실행됩니다.
💡 설치 아이콘이 안 보인다면: 주소창에 chrome://apps 입력 후 해당 앱을 찾아 실행할 수 있습니다.
Android 홈화면에 추가하기 (Chrome)
1 Chrome에서 사이트 접속
Chrome 앱에서 내선번호안내 사이트에 접속합니다.
2 메뉴 열기
화면 오른쪽 상단의 ⋮ (점 3개) 메뉴를 탭합니다.
🔒

⋮
3 "홈 화면에 추가" 선택
메뉴에서 "홈 화면에 추가" 또는 "앱 설치"를 탭합니다.

iPhone / iPad 홈화면에 추가하기 (Safari)
1 Safari에서 사이트 접속
반드시 Safari를 사용하세요. Chrome/다른 브라우저에서는 홈화면 추가가 안 됩니다.
2 공유 버튼 탭
화면 하단의 공유 버튼 (⬆ 네모+화살표)을 탭합니다.
extension-master.vercel.app
◁ ▷ ⬆ ☐ ⊞
3 "홈 화면에 추가" 선택
공유 메뉴를 위로 스크롤하여 "홈 화면에 추가"를 탭합니다.
메시지
메일
───────────
즐겨찾기에 추가
➕ 홈 화면에 추가
읽기 목록에 추가
4 이름 확인 후 추가
오른쪽 상단의 "추가"를 탭합니다.
5 완료!
홈화면에서 앱처럼 전체화면으로 실행됩니다.
